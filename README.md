# Eternal Rickroll

![](god%27s%20will.png)

This program outputs lines based on this chart.

It will stop based on line count.

After the generated lines, stats are displayed.

# Usage

```txt
$> ./eternal_rickroll
     (Program generates 69420 lines, and stats.)
$> ./eternal_rickroll 10
     (Program generates 10 lines, and stats.)
$> ./eternal_rickroll -1
     (ETERNAL MODE)
```
